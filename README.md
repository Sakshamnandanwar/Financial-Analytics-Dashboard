# Automated Data Processing Pipeline

This project focuses on building an automated pipeline to clean, transform, and prepare raw data for analysis.

## Objective
Automate repetitive data processing tasks to reduce manual effort and ensure consistent, clean datasets.

## Tools & Technologies
- Excel
- Data Cleaning & Exploratory Data Analysis
- Data Visualization

## What I Built
- Automated data ingestion from raw files
- Implemented data cleaning and transformation steps using Pandas
- Created a reusable pipeline to generate analysis-ready datasets
- Validated data quality before output

## Outcome
- Reduced manual data handling
- Improved data consistency and processing efficiency
